# File Organizer

A Python automation tool that organizes files into categorized folders based on their file extensions.

## Features

- Organizes Images, Documents, Music, Videos, Archives, and more
- Automatically creates folders when needed
- Works with any folder path
- Simple command-line interface

## Supported Categories

- Images (.jpg, .png, .jpeg)
- Documents (.pdf, .docx, .txt)
- Music (.mp3)
- Videos (.mp4, .avi, .mkv)
- Archives (.zip, .rar)
- Python Files (.py)
- Spreadsheets (.xlsx, .csv)

## Usage

```bash
python organizer.py
```

Enter the folder path when prompted.

## Technologies

- Python
- os
- pathlib
- shutil
